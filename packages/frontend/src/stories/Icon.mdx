# Icons

#

## How to use:

You can use the following icons using the Icon component or the component with the name of the Icon. Example:

`<Icon name:"BankIcon" />`
or also
`<BankIcon />`

Some icons have properties. The properties are indicated in the icon name in the following list.

ArrowIcon is the only icon with the property direction that allows values of "up", "down", "left", and "right". For the other icons, you can simply add fill or hidden if it is indicated in the icon name in the following list.

`<Icon name:"BankIcon" fill />`
or also
`<BankIcon fill /> `

`<Icon name:"ArrowIcon" direction="up">`
or also
`<ArrowIcon direction="up">`

If you want to lazy de load of the icons, please use `<IconLazy name:"BankIcon" >` instead.
